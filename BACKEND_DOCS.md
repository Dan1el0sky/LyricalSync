# Backend Documentation

The LyricalSync backend provides an API for searching YouTube Music, downloading songs, fetching lyrics, and running word-level audio alignment.

## Core Stack
- **Framework:** FastAPI
- **Audio Retrieval:** ytmusicapi & yt-dlp
- **Lyrics Fetching:** Musixmatch & LRCLIB
- **Transcription/Alignment:** stable-ts (Stable Whisper)

## Execution Flow (`/api/process`)
1. **Download**: Grabs the best audio stream using `yt-dlp` and extracts it as an `.mp3`.
2. **Fetch Lyrics**: Queries Musixmatch first. If blocked/failed, queries LRCLIB. LRCLIB provides highly accurate synced LRC lyrics, which are then passed to the aligner.
3. **Alignment**:
   - If LRC timestamps are provided, they strictly override Whisper's alignment to guarantee chorus/echo syncing.
   - If plain text is provided, `stable-ts` uses DTW (Dynamic Time Warping) to force-align the text to the audio file.
   - If no text is provided, `stable-ts` transcribes the audio from scratch and generates word-level timestamps.
4. **Processing**: Instrumentals (gaps > 5s) are injected with a `🎵` symbol. Forward/backward heuristic padding fixes skipped words and captures long held notes.
5. **Output**: The frontend downloads the generated MP3 from the static file server and receives the JSON alignment array.

## Endpoints

### 1. `POST /api/search`
**Description:** Searches YouTube Music for a song.
**Request Body:**
```json
{
  "query": "Never Gonna Give You Up"
}
```
**Response:**
```json
{
  "songs": [
    {
      "videoId": "dQw4w9WgXcQ",
      "title": "Never Gonna Give You Up",
      "artists": ["Rick Astley"],
      "album": "Whenever You Need Somebody",
      "duration": 213,
      "thumbnail": "url..."
    }
  ]
}
```

### 2. `POST /api/process`
**Description:** Initiates download and processing.
**Request Body:**
```json
{
  "video_id": "dQw4w9WgXcQ",
  "title": "Never Gonna Give You Up",
  "artist": "Rick Astley"
}
```
**Response:**
Returns the complete alignment payload, including the source used, character timings, and word timings.

### 3. `GET /api/progress/{video_id}`
**Description:** Polls the current task progress. Returns a status string and completion percentage.

### 4. `DELETE /api/downloads`
**Description:** Cleans up the root directory of any cached `.mp3` files generated by `yt-dlp`.

### 5. `GET /api/settings` and `POST /api/settings`
**Description:** Manages custom API tokens (Musixmatch / Genius) stored in the `.env` file.
